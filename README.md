# Navigation Project - React Native

## Description
React Native project focused on implementing navigation using React Navigation. It includes options for bottom tabs, side drawer, and material top tabs.

## Installation
1. **Node.js:** Ensure you have Node.js installed (version 18 or higher).
2. **Dependencies:** Run `npm install` to install the dependencies.
3. **Android:** To run on Android, use `npm run android`.
4. **iOS:** To run on iOS, use `npm run ios`.

Explore and enjoy navigating within the application!
